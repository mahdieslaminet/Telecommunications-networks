این مخزن شامل مطالب، مستندات و پیاده‌سازی‌های مرتبط با شبکه‌های مخابراتی و بی‌سیم با تمرکز بر شبکه‌های ارتباطی پهپادها (UAV Networks) است. موضوعات مطرح‌شده شامل شبیه‌سازی شبکه، پروتکل‌های مسیریابی و MAC، مدل‌های تحرک، ارزیابی عملکرد و تحلیل رفتار شبکه در سناریوهای پویا بوده و بر اساس مطالعات و مقالات پژوهشی، از جمله UavNetSim-v1، گردآوری شده‌اند.
